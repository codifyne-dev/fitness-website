# Lofi Music Tracks

This directory should contain royalty-free lofi hip-hop tracks for the background music player.

## Recommended Sources:

### 1. Pixabay Music (Free, No Attribution Required)
- Visit: https://pixabay.com/music/search/lofi/
- Download 3-5 lofi tracks
- Rename them as: `lofi-track-1.mp3`, `lofi-track-2.mp3`, etc.

### 2. Freesound.org (Creative Commons)
- Visit: https://freesound.org/browse/tags-sounds/?q=lofi
- Look for tracks with CC0 or CC BY licenses
- Download and rename as above

### 3. Bensound (Free with Attribution)
- Visit: https://www.bensound.com/royalty-free-music/lofi
- Download tracks like "Sway", "Break The Disc", "Shelter"
- Rename as: `lofi-track-1.mp3`, `lofi-track-2.mp3`, etc.

### 4. YouTube Audio Library
- Visit: https://studio.youtube.com/channel/UC/music
- Search for "lofi" or "chill"
- Download and rename as above

## File Requirements:
- Format: MP3
- Length: 2-5 minutes recommended
- Quality: 128kbps or higher
- License: Royalty-free or Creative Commons

## File Naming:
Place the downloaded files in this directory with these exact names:
- `lofi-track-1.mp3`
- `lofi-track-2.mp3` 
- `lofi-track-3.mp3`

## License Notes:
- Always check the license terms before using
- Some sources require attribution in your project
- CC0 (Public Domain) is the most permissive
- CC BY requires attribution but is free to use

## Alternative: Use Generated Music
If you prefer to keep the current generated music instead of real tracks, you can delete this README and the component will fall back to the Web Audio API generated sounds.
